Tentang Aplikasi

Aplikasi ini menggunakan Google Text-to-Speech (unofficial API) untuk mengonversi teks menjadi suara.

Fitur:

  • Support multiple languages
  • Pengaturan kecepatan suara
  • Riwayat konversi
  • Download audio
  • Responsive design

Teknologi:

  • Backend: Express.js
  • Frontend: Alpine.js
  • TTS: Google Translate TTS API

Cara Penggunaan

Langkah-langkah:

  1. Masukkan teks di kolom input (maksimal 200 karakter)
  2. Pilih bahasa yang diinginkan
  3. Atur kecepatan suara (0.5x - 2.0x)
  4. Klik "Konversi ke Suara"
  5. Putar audio hasil konversi

Tips:

  • Teks panjang akan dipotong otomatis
  • Gunakan bahasa yang sesuai dengan teks
  • Kecepatan 1.0x adalah kecepatan normal
  • Riwayat tersimpan di browser Anda
